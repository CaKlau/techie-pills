# 💊 Techie Pills

Code snippets, scripts, and configuration files from YouTube videos - bite-sized technical content covering coding, AI, algorithms, Kubernetes, scripting, and more.

## 📺 About

This repository is the companion to the official [Techie Pills YouTube channel](https://www.youtube.com/@techiepills), where we share practical, hands-on content around cloud-native infrastructure, automation, and developer tooling. Each folder corresponds to a video or series, so you can follow along or grab what you need directly.

## 🗂️ Structure

```
techie-pills/
├── topic1/
├── topic2/
└── topic3/
```

Folders and topics grow with each new video - check back regularly.

## 🚀 Usage

Clone the repo and navigate to the folder matching the video you're watching:

```bash
git clone https://github.com/your-username/techie-pills.git
cd techie-pills/<topic>
```

Each subfolder contains a short `README.md` or inline comments explaining the context.
